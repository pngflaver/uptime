# **App Name**: NodeVigil

## Core Features:

- Node List Display: Display a list of configured nodes, each with its current status (online/offline).
- Ping Test: Regularly ping each node to determine its status, indicated visually.
- Ping Graph: Display a graph showing ping response times for each node over time.
- Add Node: Allow users to add new nodes by specifying their IP address or domain name.
- Remove Node: Enable users to remove existing nodes from the monitoring list.
- Adjust Ping Interval: Provide a setting for users to adjust the frequency of ping tests.
- Script Segregation: Ensure all scripts are kept within their own distinct canvas to maintain modularity.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke trust and stability in the network monitoring.
- Background color: Light gray (#ECEFF1), a muted tone that is easy on the eyes for extended dashboard use.
- Accent color: Teal (#009688) to highlight active elements and status indicators.
- Body and headline font: 'Inter' sans-serif for a clean and modern interface, readable at various sizes.
- Code Font: 'Source Code Pro' for displaying code snippets.
- Use simple, clear icons to represent node status and actions.
- Subtle transitions and animations to provide feedback on node status changes.
